# Databasejava - Muhamad Abdullah Farhan - I.2510738

tugas mata kuliah bahasa pemrograman yang mengintegrasikan java dengan basis data (database)

## Deskripsi 

Proyek ini dibuat untuk mengelola data karyawan menggunakan bahasa pemrograman java dan juga mysql sebagai nsistem yang me manajemen basis data nya. dan syarat yang diperlukan untuk proyek ini diantaraya :

- **Java Swing** untuk antarmuka grafis
- **MySQL** sebagai database
- **Maven** untuk manajemen dependensi dan build




## struktur folder 
- src/ : source code program
- database.sql : file export database (terletak di root)
- README.md : Dokumentasi tugas
